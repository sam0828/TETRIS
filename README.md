# 俄羅斯方塊遊戲 🎮

一個採用測試驅動開發（TDD）方法開發的俄羅斯方塊遊戲，使用 TypeScript、Vite 和 Vitest。

## 功能特性

- ✅ 完整的俄羅斯方塊遊戲機制
- ✅ 7 種不同的方塊形狀（I、O、T、S、Z、L、J）
- ✅ 方塊旋轉和碰撞檢測
- ✅ 行清除機制和分數計算
- ✅ 動態難度調整（隨著清除的行數增加難度）
- ✅ 實時遊戲分數、行數、等級顯示
- ✅ 完整的單元測試覆蓋（27 個測試）
- ✅ GitHub Pages 自動部署
- ✅ 響應式 UI 設計

## 技術棧

- **語言**: TypeScript
- **構建工具**: Vite 5
- **測試框架**: Vitest
- **部署**: GitHub Actions → GitHub Pages
- **開發方法**: 測試驅動開發 (TDD)

## 快速開始

### 安裝依賴

```bash
npm install
```

### 開發模式

```bash
npm run dev
```

在瀏覽器中打開 `http://localhost:5173` 查看遊戲。

### 運行測試

```bash
# 運行所有測試
npm test

# 運行測試（監視模式）
npm test -- --watch

# 運行測試 UI
npm run test:ui

# 生成測試覆蓋率報告
npm run test:coverage
```

### 構建生產版本

```bash
npm run build
```

構建結果將在 `dist/` 目錄中。

## 遊戲說明

### 控制方式

| 按鍵 | 操作 |
|------|------|
| ⬅️ 左箭頭 | 左移方塊 |
| ➡️ 右箭頭 | 右移方塊 |
| ⬇️ 下箭頭 | 加速下落 |
| ⬆️ 上箭頭 | 旋轉方塊 |
| 空格 | 快速下落到底部 |

### 遊戲機制

1. **方塊生成**: 遊戲開始時，會隨機生成一個方塊在棋盤頂部
2. **移動和旋轉**: 使用鍵盤控制方塊的移動和旋轉
3. **堆積**: 方塊接觸到已堆積的方塊或底部時會被鎖定
4. **行清除**: 當一行完全被填滿時，該行會被清除，上面的行下落填補
5. **分數計算**: 
   - 清除 1 行: 40 × (等級 + 1) 分
   - 清除 2 行: 100 × (等級 + 1) 分
   - 清除 3 行: 300 × (等級 + 1) 分
   - 清除 4 行: 1200 × (等級 + 1) 分
6. **難度提升**: 每清除 10 行，等級提升 1 級，遊戲速度加快
7. **遊戲結束**: 當新方塊無法生成時（棋盤頂部被填滿），遊戲結束

## 測試驅動開發（TDD）流程

本項目遵循 TDD 方法：

1. **編寫測試**: 首先編寫測試用例，描述期望的行為
2. **實現功能**: 編寫最小的代碼使測試通過
3. **重構**: 優化代碼，確保測試仍然通過

所有功能都由測試驅動，確保代碼質量和可靠性。

## GitHub Actions 部署

該項目完全自動化了 GitHub Actions 部署流程：

1. **測試**: 每次推送時自動運行所有測試
2. **構建**: 測試通過後自動構建生產版本
3. **部署**: 將構建結果自動部署到 GitHub Pages

部署工作流定義在 `.github/workflows/deploy.yml` 中。
